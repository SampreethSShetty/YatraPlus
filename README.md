# 🧭 YatraPlus
## 🚀 AI-Powered Tourist Guide Application

---

## 🌟 About the Project
**YatraPlus** is an AI-integrated Android application designed to provide real-time, location-based travel assistance.  
It helps users explore nearby tourist spots, hotels, temples, and transport options in a single platform.

---

## ✨ Features
- 📍 Nearby Tourist Spot Discovery  
- 🏨 Hotel Finder (Location-based)  
- 🛕 Temple Locator  
- 🚖 Cab Booking Interface  
- 🗺️ Google Maps Integration  
- 🔐 Firebase Authentication  
- ☁️ Real-time Database  
- 🤖 Basic AI Recommendations  

---

## 🛠️ Tech Stack

### 📱 Frontend
- Kotlin  
- Jetpack Compose  

### ☁️ Backend
- Firebase Authentication  
- Firebase Realtime Database  
- Firebase Cloud Storage  

### 🌍 APIs & Services
- Google Maps API  
- Fused Location Provider  

---

## 🏗️ System Architecture
User (Mobile App)
↓
Jetpack Compose UI
↓
Application Logic (Kotlin)
↓
Firebase Backend (Auth + Database + Storage)
↓
Google Maps API & Location Services

## ⚙️ Installation

```bash
git clone https://github.com/your-username/yatraplus.git
```
## Steps:

Open in Android Studio

Add google-services.json

Enable Firebase Authentication & Database

Run the application

## 🔑 Permissions
```bash
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION"/>
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION"/>
<uses-permission android:name="android.permission.INTERNET"/>
```
## ⚠️ Limitations
Basic AI recommendation logic

Requires internet connection

Cab booking is UI-based (no real API)
